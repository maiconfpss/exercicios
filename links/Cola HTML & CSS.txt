     DISPLAY é como o elemento se comporta socialmente.

<!--

    POSITION é onde o elemento vive no mundo.

        static padrão, nem pensa

        relative permite mexer nele sem perder a casa

        absolute sai da matrix, depende do pai

        fixed grudado na tela tipo anúncio chato

        sticky meio indeciso, cola quando precisa

            top / bottom / left / right
        Só funcionam se o position deixar.
        São as coordenadas tipo GPS do elemento.
-->

<!--

    BORDER é a linha em volta. Pode ser invisível, feia ou estilosa.

        ex= border-top-left-radius: 50%;

-->

<!--

    BOX-SIZING esse aqui muda vidas.

        content-box padrão caótico

        border-box o correto, civilizado, recomendado
